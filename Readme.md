# Dealzaro backend

## Tech Stack

- Node.js
- Express.js
- Mongoose
- Mongodb
- Nodemon
- dotenv
- bcrypt
- jsonwebtoken
- cors
- multer
- validator
- stripe
- razorpay
- cloudinary

## Setup

- Clone the repository
- Install dependencies
- Set up environment variables
- Run the server

## Usage

- Start the server
- Access the API at `http://localhost:4000`
